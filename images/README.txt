                    CARA MENAMBAHKAN GAMBAR
-----------------------------------------------------------------

1. Siapkan foto/gambar Anda (disarankan format JPG atau PNG)
2. Rename/ganti nama file gambar sesuai daftar di atas
3. Copy paste gambar ke folder "images" ini
4. Website akan otomatis menampilkan gambar tersebut

-----------------------------------------------------------------
                    CATATAN
-----------------------------------------------------------------

- Jika gambar tidak ditemukan, website akan menampilkan inisial 
  nama (seperti AF, BS, CR, dll) sebagai alternatif
- Ukuran yang disarankan:
  * Foto siswa: 80x80 px atau lebih (bisa square/kotak)
  * Foto wali kelas: 120x120 px atau lebih
  * Foto galeri: 400x300 px atau lebih (aspect ratio 4:3)
